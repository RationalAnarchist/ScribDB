Interactive Documentation

Scrollarr provides a fully interactive REST API documentation using Swagger UI and ReDoc. You can test endpoints directly from the browser.

Open Swagger UI Open ReDoc

Examples

Get System Status

curl -X GET "http://localhost:8000/api/status"

Search for a Story

curl -X GET "http://localhost:8000/api/search?query=mother+of+learning"

Add a New Story

curl -X POST "http://localhost:8000/api/add" \
-H "Content-Type: application/json" \
-d '{"url": "https://www.royalroad.com/fiction/21220/mother-of-learning", "profile_id": 1}'
