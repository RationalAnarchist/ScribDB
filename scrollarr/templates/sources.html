Configure Source

AO3 often requires authentication or specific cookies to bypass Cloudflare.

  1. Log in to AO3 in your browser.
  2. Open Developer Tools (F12) -> Application/Storage -> Cookies.
  3. Copy the values for _session_id and user_credentials (if logged in).
  4. Format as a JSON object below.

Example Configuration:

{"cookies": "_session_id=...; user_credentials=..."}

Enter configuration as JSON key-value pairs.
